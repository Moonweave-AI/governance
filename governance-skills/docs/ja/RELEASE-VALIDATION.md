# リリース検証

## 0.1.0（プレリリース）

本リリースは Moonweave Governance をベンダーニュートラルな Skill コア・23 の焦点を絞った Skill・23 のコマンド・24 のテンプレート・決定論的 CLI、および Cursor・Codex・Claude Code・OpenCode・Kilo・Antigravity 向けプラットフォームアダプタへコンパイルする。

### 完了した検証

| 項目 | 結果 |
|---|---:|
| Node のユニットおよび統合テスト | 17 / 17 合格 |
| Agent Skill 静的チェック | 23 Skill、0 error、0 warning |
| ルーティングとトリガの静的評価 | 82 / 82 合格 |
| プラットフォームマニフェストとアダプタチェック | 93 / 93 合格 |
| リポジトリ自己監査 | blocker / high / medium / low / info がすべて 0 |

検証カバレッジ：

- 監査は被監査リポジトリのコードを実行しない；
- Secret 検出は値をマスキングし、報告書に完全な資格情報をエコーしない；
- 具現化プロジェクトで Hazard Analysis または E-Stop の証拠が欠落している場合はブロックする；
- 本番書き込み権限・Agent 権限昇格・高リスクの物理动作は人間確認へエスカレーションする；
- すべてのプラットフォームで copy インストールと安全なアンインストールをテストする；
- symlink モードは明示的に選択された場合のみ有効になる；
- インストーラは `--force` なしで既存の Skill を上書きしない；
- アンインストーラはインストール後にユーザが変更したファイルを保持する；
- すべての Skill に正負のトリガ回帰ケースがある。

### 未完了の外部検証

以下は実際のリリース環境で実施する必要があり、ローカル検証結果で代用してはならない：

- 6 つのインタラクティブ Agent 製品で実際のモデルによるペア評価を実施する；
- Moonweave AI の npm アカウントでパッケージを公開する；
- 各プラットフォームのマーケットプレースへ提出し、人手または自動審査を通過する；
- リリース CI で署名・OIDC provenance・不変の release artifact を生成する；
- 実プロジェクトでトリガ Precision / Recall・ガバナンス漏れ・人間 Review 時間・Skill 振る舞い制約カバレッジを収集する。

機械可読な結果はルートの `release-manifest.json`、詳細な方法論は `docs/EVALUATION.md` を参照。
