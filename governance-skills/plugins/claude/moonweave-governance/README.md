# Moonweave Governance for Claude Code

Moonweave AI governance, planning, RFC, engineering, security, quality, release, research, and knowledge management skills.

## Installation

```bash
/plugin marketplace add Moonweave-AI/governance
/plugin install moonweave-governance@moonweave-ai
```

- Authoritative governance source: https://github.com/Moonweave-AI/governance
- Vendor-neutral skills source: the `governance-skills/skills/` directory of the governance repo
- High-risk actions remain constrained by platform permissions, CI, security gates, and human confirmation.
