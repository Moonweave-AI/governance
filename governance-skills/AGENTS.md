# Moonweave Governance Skills Repository

This repository compiles Moonweave Governance into cross-platform Agent Skills, commands, templates, a deterministic CLI, and native plugin adapters.

## Working rules

- Before modifying a skill, cross-check `core/governance-map.json` and the upstream canonical governance spec.
- `skills/` is the vendor-neutral semantic source; `plugins/`, `.claude-plugin/`, `.cursor-plugin/`, `.agents/plugins/` are adapter outputs generated by `npm run build:adapters`.
- Do not hand-edit generated native plugin copies. Modify the source skill, command, or rule, then rebuild.
- Run `npm run verify` before committing.
- Do not weaken Stop-Ship, human-confirmation, least-privilege, provenance, or evidence-honesty requirements.
- Do not commit real secrets, personal data, or unauthorized assets.
